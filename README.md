# Easy AWS

A guide to deploy an aplication to a EC2 instance with ease.

Video on YouTube showing the process [here](https://youtu.be/iyiANe9Eszs).

## Installation

```bash
$ yarn install
```

## Copy .env.example

```bash
$ cp .env.example .env
```

## Run

```bash
$ yarn start
```

## Run with Docker

```bash
$ yarn start:docker
```
